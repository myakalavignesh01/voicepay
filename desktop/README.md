# VoicePay Desktop

Professional desktop application for VoicePay payment platform

## Features
✅ User authentication  
✅ Dashboard with stats  
✅ Payment management  
✅ Security settings  
✅ Cross-platform support  

## Installation

```bash
cd desktop
npm install
```

## Running

```bash
npm start
```

## Building

```bash
npm run build
```
